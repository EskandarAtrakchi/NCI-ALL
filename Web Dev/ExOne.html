Web Application Development

Aims

  • Extend the functionality of Web pages using client-side scripting languages.
  • Enable students to understand and develop responsive, future proof and scalable applications. Introduce different web-based languages.
  • Critique frameworks, tools, languages, and controls available for developing a rich Internet application.
  • Implement server-side scripting, and develop advanced-server side functionality.
  • Evaluate the Pros and Cons of using client-side scripting versus server-side scripting.

Learning Outcomes

  1. Manipulate and define data structures using a mark-up language. Implement server-side scripting and develop advanced server-side functionality.
  2. Evaluate the Pros and Cons of using client-side scripting versus server-side scripting
  3. Critique frameworks, tools, languages, and controls available for developing a rich Internet application.
  4. Extend the functionality of Web pages using client-side scripting languages.
